# 🎯 초성 이름 맞추기 게임

팀원들의 이름을 초성으로 맞춰보는 재미있는 게임!

## 👥 팀 소개

| 이름 | 역할 |
|------|------|
| 김선숙 | 팀장 |
| 하승호 | 팀원 |
| 민경환 | 팀원 |
| 홍주희 | 팀원 |
| 김재원 | 팀원 |
| 조현규 | 팀원 |

## 🎮 게임 방법

1. `index.html` 파일을 브라우저에서 열어주세요.
2. 화면에 표시되는 초성을 보고 팀원의 이름을 맞춰보세요.
3. 정답을 입력하고 제출 버튼을 누르거나 Enter 키를 누르세요.
4. 총 6명의 팀원을 모두 맞추면 게임 종료!

## 🚀 실행 방법

```bash
# 저장소를 클론한 후, 브라우저에서 index.html 파일을 열어주세요.
# 또는 로컬 서버를 실행합니다:
python -m http.server 8000
# 그리고 http://localhost:8000 에 접속
```

## 🛠️ 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
