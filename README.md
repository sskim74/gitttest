# 🎸 서태지 팬덤 테스트

당신은 서태지를 얼마나 알고 있나요? 10문제로 확인핸보세요!

## 🚀 실행 방법

**`index.html` 파일 하나만 다운로드해서 브라우저에서 열기만 하면 됩니다!**

```
1. GitHub에서 index.html 파일 다운로드
2. 더블클릭해서 브라우저에서 실행
3. 끝!
```

> ⚠️ **중요**: CSS와 JavaScript가 모두 HTML 파일에 포함되어 있어서 별도 파일이 필요 없습니다.

## 🎮 게임 방법

1. 브라우저에서 `index.html` 파일 열기
2. 객관식 또는 주관식 문제에 답핸보세요
3. 총 10문제를 모두 풀면 등급이 결정됩니다!
